# SFTP 教學

SFTP 是檔案傳輸協議，您可使用 SFTP 上傳大型檔案。

## 如何使用 SFTP

<Steps>
### 下載 SFTP 工具
首先，您需要有一個 SFTP 連接器，以下為建議的 SFTP 連接工具：
1. WinSCP: https://winscp.net/eng/download.php
2. FileZilla: https://filezilla-project.org/download.php?type=client

---

### 配置 SFTP 連線資訊

我們以 WinSCP 為例：
![配置 SFTP](/panel/sftp/config.png)

1. 主機名稱與連接埠
| 節點       | 連線位置                                       |
| ---------- | ------------------------------------------ |
| TW-P   |  sftp://sftp.coffeehost.net:2023                |
| TW-S |    sftp://sftp.coffeehost.net:2024                    |
| MY-1       | sftp://sftp.coffeehost.net:2022                 |
![連線資訊](/panel/sftp/information.png)

2. 使用者名稱及密碼

請進入您的伺服器的「Settings」頁面


- 使用者名稱：控制面板上的資訊（框起來的）
- 密碼：登入控制面板的密
